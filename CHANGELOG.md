# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.1.0] - 2024-01-20

### Added
- GitHub Actions CI/CD workflows for automated testing and releases
- Comprehensive README with setup instructions and architecture overview
- CONTRIBUTING.md guide for developers
- MIT License
- .gitignore file for excluding build artifacts and dependencies
- .env.example with documented environment variables

### Changed
- Improved documentation for project setup and usage
- Enhanced .env.example with better variable descriptions

### Fixed
- Better structured project configuration files

## [1.0.0] - 2024-01-15

### Added
- Initial MVP release of DEN Bot
- Telegram bot interface with multi-step conversations
- Deal creation workflow
- In-memory and MongoDB storage options
- User and deal management services
- Comprehensive TypeScript setup with strict type checking
- ESLint configuration for code quality
- Command handlers for /start, /menu, /createdeal, /help

### Features
- Secure deal escrow system
- User authentication via Telegram
- Deal status tracking
- Input validation
- Middleware-based logging system
- Inline and static keyboard support
